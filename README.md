# 🌸 Love You Mom — Maa Ki Kahani

> *An interactive digital memory book for the most important woman in your life.*

Built for the **BuiltWithTRAE: Mother's Day Edition** challenge.

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🏠 **Hero Landing** | Beautiful animated landing with flower petals and feature showcase |
| 🕰️ **Interactive Timeline** | Chronicle mom's life journey with photos, dates & stories |
| 🍳 **Recipe Vault** | Preserve her signature recipes with Cook Along mode |
| 🎵 **Voice Library** | Record & play her voice, lullabies, and stories |
| 💌 **Wisdom Wall** | Her advice and life lessons as beautiful cards |
| 📸 **Memory Gallery** | Photo gallery with masonry layout and lightbox |
| 🌙 **Bedtime Mode** | Soothing dark theme with starry night and lullabies |
| 🎁 **Shareable Card** | Generate beautiful Mother's Day cards to download |
| 💌 **Love Letter Composer** | Write heartfelt letters with templates & typewriter effect |
| 🙏 **Gratitude Journal** | Daily gratitude entries with streak tracking |
| 🧩 **Mom's Favorites Quiz** | "How Well Do You Know Your Mom?" interactive quiz |
| 🌳 **Family Tree** | Visual interactive family tree builder |
| 🤝 **Promise Wall** | Make & track promises to mom with confetti celebrations |
| 🎉 **Celebration Countdown** | Live countdown timers for Mother's Day & birthdays |

## 🌍 Bilingual
Full **English** & **Bangla (বাংলা)** support with one-click toggle.

## 🛠 Tech Stack
- **Next.js 15** (App Router)
- **React 19**
- **Tailwind CSS 4**
- **Framer Motion**
- **html2canvas**
- **Web Audio API**
- **LocalStorage** persistence (no backend needed)

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000)

## 📦 Build

```bash
npm run build
npm start
```
